Pewarisan di CSS adalah konsep di mana elemen HTML bisa mewarisi beberapa nilai properti dari elemen induknya (parent).

‚ö†Ô∏è Tidak semua properti diwariskan secara otomatis! Beberapa properti harus disetel secara manual menggunakan inherit.

Apa yang Bisa Diwariskan?

Contoh properti yang umum diwariskan:

  • color
  • font-family
  • letter-spacing
  • line-height
  • visibility

Contoh Kasus: Warna pada Elemen <a>

Struktur HTML:

<body>
  <h1>Judul Halaman</h1>
  <a href="#">Ini link</a>
  <p>Ini paragraf</p>
</body>

CSS awal:

body {
  font-family: Arial;
  color: darkgray;
}

Judul Halaman

Ini link

Ini paragraf

Perhatikan bahwa teks link tetap berwarna biru secara default. Ini karena <a> tidak mewarisi color.

Solusi: Gunakan inherit

Tambahkan CSS berikut agar link ikut mewarisi warna dari parent-nya:

a {
  color: inherit;
}

Hasilnya, link kini mengikuti warna teks dari elemen <body>.

Kesimpulan

  • Beberapa properti memang diwariskan secara default di CSS.
  • Namun untuk properti seperti color pada <a>, kita perlu menyetel inherit secara eksplisit.
  • Ini membantu menjaga konsistensi tampilan ketika style induk berubah.
